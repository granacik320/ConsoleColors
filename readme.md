
# ConsoleColors

Beautiful color gradients in your devtools ConsoleColors




# Install

Just use a CDN

```html
<script type="text/javascript" src="..."></script>
```


## Usage

```javascript
console.log(...gradient('add some text',"#cc4e4e","#e17d59"))
```
### Multi line gradients

```javascript
console.log(...gradient([
    '██╗██████╗░',
    '╚═╝╚════██╗',
    '░░░░█████╔╝',
    '░░░░╚═══██╗',
    '██╗██████╔╝',
    '╚═╝╚═════╝░',
],"#ff00d5","#45acec"))
```
